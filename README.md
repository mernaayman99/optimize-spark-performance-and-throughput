# Optimize Spark Performance and Throughput

This repository contains hands-on notebooks, datasets, Spark UI analysis examples,
and optimization exercises used in the course **Optimize Spark Performance and Throughput**.

## What You Will Learn
- How to analyze Spark jobs using the Spark UI
- How to identify bottlenecks such as shuffle pressure and data skew
- How to tune executors, memory, and partitions to meet SLAs
- How to apply Spark optimization techniques systematically

## Repository Structure
- Module 1: Spark UI Analysis
- Module 2: Data Skew Optimization
- Module 3: SLA-driven Tuning
- Course-End Project: End-to-End Spark Job Optimization

## Tools Used
- Apache Spark
- Databricks (Free Edition compatible)
- Spark UI
- Jupyter Notebooks

This repository is designed to work even with **Databricks Free Edition**.
